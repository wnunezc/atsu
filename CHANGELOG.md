# Changelog

## 1.0.0 - 2026-06-18

- Initial public version.
- Manifest V3 extension with restricted Stack Exchange host permissions.
- Per-site configuration stored in `chrome.storage.local`.
- Diagnostics mode in the popup and ATSU sidebar panel.
- Smarter comment templates for questions, answers, and Meta sites.
- Meta-specific behavior that avoids default technical MRE prompts.
- Compact tabbed popup UI.
- Color profiles: none, soft, medium, strong, custom.
- Settings export/import as local JSON.
- Reset all settings action.
- Local tracking for opened question IDs without the browser history permission.
- `NEW` / `NUEVO` badges for locally unvisited questions.
- `CLOSED` / `CERRADA` badges when explicit closed or duplicate markers are available.
- Privacy policy and Chrome Web Store preparation notes.
